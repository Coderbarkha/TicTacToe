# 🎮 Tic Tac Toe Game

A fun and interactive **Tic Tac Toe** game built using **HTML**, **CSS**, and **JavaScript**. This project allows two players to play the classic game directly in the browser with a clean UI and smooth gameplay.

---

## 🛠️ Technologies Used

- ✅ HTML5
- 🎨 CSS3 (Flexbox for layout)
- 🧠 JavaScript (DOM Manipulation & Game Logic)

---

## ✨ Features

- 🔹 Two-player mode (Player X and Player O)
- 🔹 Highlighting winning combinations
- 🔹 Restart/Reset button to play again
- 🔹 Basic responsive design (mobile-friendly)
